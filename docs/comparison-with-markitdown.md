# Comparison with Microsoft MarkItDown

Microsoft MarkItDown is the core converter. MarkItDown Plus is a companion toolkit for batch and AI-pipeline workflows.

## Use Microsoft MarkItDown when

- You only need to convert one file.
- You want the official core converter directly.
- You are building your own custom pipeline from scratch.

## Use MarkItDown Plus when

- You need to convert a folder of files.
- You want recursive scanning and extension filters.
- You want cleaned Markdown output.
- You want JSONL chunks for RAG or vector database ingestion.
- You want a manifest showing what converted successfully and what failed.
